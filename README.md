### 🚀 Featured Projects

#### 🚗 [PayModule Parking](https://github.com/pedrocamposlive/paymodule-parking)
A complete SaaS platform for recurring parking lot billing. Built with a modern stack (TypeScript, React, Node.js), it features automated Pix generation via MercadoPago and WhatsApp integration via Evolution API.

#### 🎬 [Ketter 3.0](https://github.com/pedrocamposlive/ketter3)
A production-grade file transfer system designed for media workflows (like Pro Tools sessions). Features triple SHA-256 verification, smart folder handling, and a robust architecture (FastAPI, React, PostgreSQL, Redis).

#### ♠️ [Project Pocket Aces](https://github.com/pedrocamposlive/pocketacesproject)
A web application for managing home poker games. Built with Python (Flask), it tracks buy-ins, stacks, and generates Pix payloads for easy financial settlements among friends.

#### 🤖 [SeraSender](https://github.com/pedrocamposlive/SERASENDER)
A 100% hands-free automation tool for uploading OCF files to the MediaShuttle portal, utilizing Selenium and PyAutoGUI for visual-based interactions.

#### 🧊 [OBJ Project](https://github.com/pedrocamposlive/OBJ_Project)
A LiDAR point cloud processing pipeline using Open3D. Includes scripts for alignment, segmentation, and 3D mesh reconstruction.
